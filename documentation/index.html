Requirements

Hello, thank you for making the purchase of FacebookPub.

To make installation of the system is required to have the following requisites:

  • PHP 5.4+
  • Cron Jobs
  • Support cURL

Installation

To install go to the folder where the files of facebookPub /install. (example: http://www.yourwebsite.com/install).

Follow the installation steps informing the data of access to your database, login of cadastre, purchase code, etc.

After installation delete or rename the folder install for your own safety

CronJobs

The system only works if cronjobs is correctly configured. Follow the commands below, set them to run from 1 to 5 minutes:

wget --spider -O - http://yourwebsite.com/cron/posts >/dev/null 2>&1
wget --spider -O - http://yourwebsite.com/cron/likes_page >/dev/null 2>&1
wget --spider -O - http://yourwebsite.com/cron/pages_manager >/dev/null 2>&1
wget --spider -O - http://yourwebsite.com/cron/groups_manager >/dev/null 2>&1

Facebook Permissions

The system works with some facebook permissions, if necessary, create your application in http://developers.facebook.com ask facebook permission to use these permissions below:

  • publish_actions
  • manage_pages
  • publish_pages
  • pages_show_list
  • public_profile
  • email
  • user_managed_groups
